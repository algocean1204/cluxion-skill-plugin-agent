# Static artifact review

Use only for a complex or print-bound artifact.

1. Content pass: exact copy, spelling, required marks, attribution.
2. Geometry pass: trim/bleed, margins, alignment, crop, bounds.
3. Hierarchy pass: one focal point, readable order, no decorative noise.
4. Output pass: target dimensions, color mode if specified, embedded/local fonts.
5. Render pass: inspect the exported PNG/PDF rather than trusting source coordinates.

An intentional overlap must have a clear layer order and must not obscure required text. Everything else counts as a defect.
