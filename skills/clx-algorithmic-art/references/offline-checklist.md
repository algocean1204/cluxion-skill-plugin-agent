# Offline generative-art gate

- HTML references only relative local files.
- Copy `assets/p5.min.js` and `assets/P5-LICENSE.txt` with p5 outputs; `assets/p5.js` is the bundled readable source. Use the Canvas template when native Canvas is enough.
- No CDN, remote font, analytics, external texture, or unapproved fetch.
- Seed and parameters reproduce the same result.
- Animation has a bounded entity count and pause/reset path.
- Output is original and does not imitate a living artist's distinctive style.
- If embedded in UI, provide a static/reduced-motion alternative.
